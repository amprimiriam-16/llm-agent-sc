# Changelog

- Initial release of SCIP QA Platform
- FastAPI backend with RAG endpoints
- Streamlit frontend with chat interface
- Azure Cosmos DB vector store integration

- Agentic RAG with query decomposition

- Comprehensive documentation

- Analytics dashboard
- Health monitoring
- BASF governance compliance
